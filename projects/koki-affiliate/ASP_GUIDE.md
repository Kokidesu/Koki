# アフィリエイト登録ガイド（最初の1回だけ・手動）

ASP（アフィリエイト・サービス・プロバイダ）への登録は、**本人確認・振込先口座・税情報**が
必須で、**自動化できません**（各社の規約でもボット登録は禁止されています）。
ただし登録は最初の1回だけ。ここを済ませれば、記事生成〜公開はすべて自動で回せます。

## 登録ハードルが低い順

### 1. もしもアフィリエイト（初心者にいちばんおすすめ）
- Amazon・楽天・Yahoo!ショッピングを**まとめて**扱える
- 審査がやさしく、「かんたんリンク」でリンク生成が楽
- W報酬制度など初報酬を得やすい仕組みがある

### 2. A8.net（案件数No.1）
- **自分のサイトが無くても登録可**
- 物販以外（無料登録・資料請求などの成果報酬）も多く、最初の1件を出しやすい

### 3. 楽天アフィリエイト
- 楽天会員ならすぐ始められ、審査はほぼなし
- 報酬は楽天ポイント中心（現金化の条件あり）

### 4. Amazonアソシエイト
- 登録後 **180日以内に3件の適格販売**が必要（条件未達だと却下）
- 単価は低めだが、商品数と知名度は最強

## 登録に必要なもの（共通）

- メールアドレス
- 氏名・住所などの本人情報
- **振込先の銀行口座**
- （報酬額により）税に関する情報
- 公開中のサイトURL → **このツールが作る GitHub Pages のURLでOK**

## 登録後にやること（ここから先は全部自動）

1. 各ASPで発行された **トラッキングID / タグ** を控える
2. `config.toml` の `amazon_tag` などに設定（または GitHub Actions の Variables に `AMAZON_TAG`）
3. 以降、生成される**全記事のリンクに自動反映**
   （CTAは表示時に差し込む設計なので、**既存記事にも遡って反映**されます）

## ⚠️ 規約の注意（必ず各ASPで確認）

- 価格や在庫の**固定的な表記の禁止**（「〇〇円」と書いて放置しない）
- **自己購入・家族購入の禁止**
- **虚偽の体験談・効果効能の断定の禁止**
- ステマ規制対応として、本ツールは「PR / 広告」表記と開示文を自動で入れています
