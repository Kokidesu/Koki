# ケース面接ドリル（会社別 模擬面接）

**志望ファームを選ぶと、その会社の出題傾向に合わせたケース問題が面接形式で出てくる**対策アプリ。
依存ライブラリなしの **`index.html` 1ファイル完結**（HTML/CSS/JS）。ブラウザで開くだけで動く。

## 使い方の流れ

1. **会社を選ぶ**（MBB＋戦略系＋総合系の12ファーム。ブランドカラーのロゴ風モノグラム付き）
2. その会社の**出題リスト**から問題を選ぶ（🎲ランダム出題も可）
3. **模擬面接スタート** — 面接官との対話形式で進行：
   - 問題提示 → **確認質問**（想定Q&A付き）
   - **構造化**（タイマー付き・模範フレーム付き）
   - **計算問題**（数値入力で⭕❌判定・解法付き）
   - **仮説出し** → **30秒の提言**（模範解答付き）
4. 終了後に**振り返りセルフチェック**で自己採点 → 次の問題へ

## コンテンツ

| 種類 | 内容 |
|------|------|
| ファーム | McKinsey / BCG / Bain / Strategy& / Monitor Deloitte / EY-P / Kearney / Roland Berger / Oliver Wyman / L.E.K. / Accenture / 総合系 |
| ケース | 収益性・市場参入・M&A・価格設定・コスト・成長・市場規模・DX・事業DD・PEI（人物面接）の全15問。各ファームに3問以上 |
| 補助ツール | フレームワーク集 ／ 暗算トレーナー（60秒×4難易度） ／ 用語集（検索付き） |

## ローカルで見る

```bash
open projects/case-interview/index.html
# or
cd projects/case-interview && python3 -m http.server 8000   # → http://localhost:8000
```
